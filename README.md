## 目录结构
* src
  * app.controller.spec.ts  // 控制器的单元测试。
  * app.controller.ts // 具有单一路由的基本控制器。
  * app.module.ts  // 应用的根模块。
  * app.service.ts  // 具有单一方法的基本服务。
  * main.ts // 使用核心函数 NestFactory 创建 Nest 应用实例的应用入口文件。

## 安装依赖
// 下载依赖  
`npm install`

// 运行依赖  
`npm run dev:start`

// 打包依赖  
`npm run build`
